# ScrollView-training

### ScrollViewの制約の手順
- UIScrollViewの位置とサイズの制約
- UIScrollViewの上にContentsViewを乗せる
- ContentsViewとContentLayoutGuideに制約をつける
- ContentsViewの上下左右をContentLayoutGuideに合わせる(慣例的に)
- ContentsViewの大きさをFrameLayoutGuideで決める
